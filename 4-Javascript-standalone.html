JavaScript

Declaración de variables: let y const

let permite declarar variables que pueden ser reasignadas. const declara constantes que no pueden cambiar su valor después de la asignación inicial.

let contador = 0;
contador = 1; // Correcto, se puede reasignar

const PI = 3.1416;
PI = 3.14; // Error: no se puede reasignar una constante

Usa const por defecto para evitar cambios accidentales, y let solo cuando necesites reasignar.

Control de flujo

if-else

Ejecuta bloques de código basados en condiciones booleanas:

if (edad >= 18) {
	console.log('Mayor de edad');
} else {
	console.log('Menor de edad');
}

switch

Útil para múltiples casos basados en el valor de una variable:

switch (dia) {
	case 'lunes':
		console.log('Inicio de semana');
		break;
	case 'viernes':
		console.log('Fin de semana laboral');
		break;
	default:
		console.log('Día normal');
}

Bucles

while

Se ejecuta mientras la condición sea verdadera:

let i = 0;
while (i < 5) {
	console.log(i);
	i++;
}

for clásico

Bucle con inicialización, condición y actualización:

for (let i = 0; i < 5; i++) {
	console.log(i);
}

for-in

Itera sobre las propiedades enumerables de un objeto:

const persona = { nombre: 'Ana', edad: 25 };
for (let propiedad in persona) {
	console.log(propiedad + ': ' + persona[propiedad]);
}

for-of

Itera sobre elementos iterables como arrays:

const frutas = ['manzana', 'pera', 'uva'];
for (let fruta of frutas) {
	console.log(fruta);
}

forEach

Método de arrays para ejecutar una función en cada elemento:

frutas.forEach(function(fruta) {
	console.log(fruta);
});
frutas.forEach(fruta => console.log(fruta));

break y continue

break detiene la ejecución de un bucle. continue salta a la siguiente iteración:

for (let i = 0; i < 10; i++) {
	if (i === 5) break;
	console.log(i);
}

for (let i = 0; i < 5; i++) {
	if (i === 2) continue;
	console.log(i);
}

Funciones

Función normal (declarada)

Se define con la palabra clave function:

function saludar(nombre) {
	return 'Hola, ' + nombre;
}
console.log(saludar('Mundo'));

Función anónima

Función sin nombre, asignada a una variable:

const saludar = function(nombre) {
	return 'Hola, ' + nombre;
};

// Uso en escuchadores
const boton = document.getElementById('miBoton');
boton.addEventListener('click', function(event) {
	console.log('Botón clickeado');
});

Función flecha

Sintaxis más concisa:

const saludar = (nombre) => 'Hola, ' + nombre;

// Uso en escuchadores
const boton = document.getElementById('miBoton');
boton.addEventListener('click', (event) => {
	console.log('Botón clickeado');
});

Esperar a que cargue el documento

Usar el evento DOMContentLoaded para ejecutar código cuando el DOM esté listo:

document.addEventListener('DOMContentLoaded', function() {
	// Aquí va el código que interactúa con el DOM
});

Esto evita errores al buscar elementos antes de que existan.

Interacción con el DOM

Métodos para obtener elementos

// Por id
const boton = document.getElementById('miBoton');

// Por etiqueta
const forms = document.getElementsByTagName('form');
const form = forms[0];

// Por clase
const cajas = document.getElementsByClassName('caja');
const primeraCaja = cajas[0];

parentNode: Acceder al elemento padre

const elemento = document.getElementById('miElemento');

// Acceder al padre inmediato
const padre = elemento.parentNode;

// Acceder al abuelo
const abuelo = elemento.parentNode.parentNode;

Objetos

Creación de objetos

Los objetos se crean usando llaves {} con pares clave-valor:

const persona = {
	nombre: 'Juan',
	edad: 30,
	ciudad: 'Madrid'
};

Acceso a propiedades

// Notación de punto
console.log(persona.nombre); // 'Juan'

// Notación de corchetes
console.log(persona['edad']); // 30

Lectura y modificación

// Modificación
persona.edad = 31;
persona['ciudad'] = 'Barcelona';

// Añadir nueva propiedad
persona.email = 'juan@example.com';

// Eliminar propiedad
delete persona.ciudad;

Almacenamiento web

localStorage

Almacena datos sin fecha de expiración:

// Guardar
localStorage.setItem('nombre', 'Juan');

// Recuperar
const nombre = localStorage.getItem('nombre');

// Guardar objeto
const usuario = { id: 1, nombre: 'Ana' };
localStorage.setItem('usuario', JSON.stringify(usuario));

// Recuperar objeto
const usuarioGuardado = JSON.parse(localStorage.getItem('usuario'));

// Limpiar
localStorage.removeItem('nombre');
localStorage.clear();

sessionStorage

Similar a localStorage, pero se elimina al cerrar la pestaña:

// El funcionamiento es idéntico a localStorage
sessionStorage.setItem('token', 'abc123xyz');
const token = sessionStorage.getItem('token');

// Guardar array
const carrito = [
	{ id: 1, producto: 'Laptop' },
	{ id: 2, producto: 'Mouse' }
];
sessionStorage.setItem('carrito', JSON.stringify(carrito));

// Recuperar
const carritoGuardado = JSON.parse(sessionStorage.getItem('carrito'));

Cookies

Pequeños fragmentos de datos con control de expiración. Se envían con cada solicitud HTTP.

localStorage y sessionStorage son APIs modernas y más convenientes para datos del lado del cliente.
