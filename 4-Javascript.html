JavaScript - Fundamentos

Principales conceptos para interactuar con el DOM y gestionar eventos.

Esperar a que cargue el documento

Usar el evento DOMContentLoaded para ejecutar código cuando el DOM esté listo:

document.addEventListener('DOMContentLoaded', function() {
	// Aquí va el código que interactúa con el DOM
});

Esto evita errores al buscar elementos antes de que existan.

Obtener referencias al DOM

Formas habituales para obtener elementos del DOM:


		// Por id (devuelve un elemento directamente)
		const boton = document.getElementById('miBoton');

		// Por nombre de etiqueta (devuelve una colección similar a una lista)
		const forms = document.getElementsByTagName('form');
		const form = forms[0]; // primer formulario en la página

		// Por clase (devuelve una colección similar a una lista)
		const cajas = document.getElementsByClassName('caja');
		const primeraCaja = cajas[0];

getElementById devuelve un único objeto Element. getElementsByTagName y getElementsByClassName devuelven colecciones (HTMLCollection) que funcionan como listas: accede por posición (por ejemplo [0]) o itera la colección para procesar todos los elementos.

Añadir escuchadores

Formas sencillas para adjuntar listeners a elementos (ejemplos usando los métodos anteriores):


// Asignación directa (propiedades .onclick, .onsubmit, ...)
const boton = document.getElementById('miBoton');
const form = document.getElementsByTagName('form')[0];

boton.onclick = function() { console.log('click'); };
form.onsubmit = function(event) { event.preventDefault(); /* ... */ };

Pros: sencillo y rápido para casos simples. Contras: solo un handler por tipo de evento.


// Alternativa moderna: addEventListener (permite múltiples handlers)
boton.addEventListener('click', miFuncion);
boton.addEventListener('click', function() { console.log('otro handler'); });

form.addEventListener('submit', miSubmitHandler);

Pros: varios listeners, opciones de captura/bubbling y posibilidad de eliminar handlers con removeEventListener.

Funciones callback y el objeto evento

Ejemplo de callback:

function onClickBoton(event) {
	// El elemento que recibió el listener (el botón en este caso)
	console.log('currentTarget:', event.currentTarget);
}

boton.addEventListener('click', onClickBoton);

Para formularios, evitar que se envíen si queremos procesar datos en cliente:

function miSubmitHandler(event) {
	event.preventDefault(); // evita el envío
	// procesar datos aquí
}
form.addEventListener('submit', miSubmitHandler);

Crear elementos

Dos formas comunes:

a) Usando innerHTML

const cont = document.getElementById('contenedor');
cont.innerHTML = '<div class="tarjeta"><p>Hola</p></div>';

b) Usando createElement y appendChild

const elemento = document.createElement('div');
elemento.classList.add('tarjeta');
elemento.innerHTML = 'Hola';
const padre = document.getElementById('contenedor');
padre.appendChild(elemento);

La segunda opción es más segura y manipulable en tiempo de ejecución. Y permite añadir escuchadores al disponer del objeto.

Gestión de atributos

Forma genérica

const img = document.createElement('img');
img.setAttribute('src', 'foto.png');
console.log(img.getAttribute('src'));

Forma específica para Clases

const caja = document.getElementById('caja');
caja.classList.add('activo');
caja.classList.remove('activo');
caja.classList.toggle('visible');
// toggle: si no está activa, la activa; si está activa, la desactiva

Forma específica para datos personalizados - Dataset

const fila = document.createElement('div');
fila.dataset.idUsuario = 123; // se guarda como 'data-id-usuario'
console.log(fila.dataset.idUsuario);

Listas en JavaScript

// Creación
const lista = [];

// Añadir
lista.push('manzana');

// Acceso
console.log(lista[0]);

// Eliminar por valor
const index = lista.indexOf('manzana');
if (index >= 0) lista.splice(index, 1);

// Recorrer
for (const elemento of lista) {
	console.log(elemento);
}

También existen métodos funcionales como find, filter...

Ejemplo práctico: Añadir items a una lista

Mini-snippet para añadir elementos desde un formulario y mostrarlos:

document.addEventListener('DOMContentLoaded', function() {
	const form = document.createElement('form');
	const input = document.createElement('input');
	input.name = 'item';
	const submit = document.createElement('button');
	submit.type = 'submit';
	submit.textContent = 'Añadir';
	const ul = document.createElement('ul');

	form.appendChild(input);
	form.appendChild(submit);
	document.body.appendChild(form);
	document.body.appendChild(ul);

	const items = [];

	form.addEventListener('submit', function(e) {
		e.preventDefault();
		const form = e.currentTarget;
		const value = e.elements.item.value.trim();
		if (!value) return;
		items.push(value);
		const li = document.createElement('li');
		li.textContent = value;
		ul.appendChild(li);
		input.value = '';
	});
});
