Introducción a HTML

¿Qué es HTML?

HTML significa HyperText Markup Language y es el estándar para crear páginas web.

Sintaxis de HTML

HTML utiliza etiquetas (tags) para estructurar el contenido. Cada etiqueta tiene un significado específico y se escribe entre corchetes angulares.

Etiquetas de Apertura y Cierre

La mayoría de las etiquetas HTML tienen una etiqueta de apertura y una de cierre:


					<etiqueta>Contenido</etiqueta>

Ejemplo:


					<p>Este es un párrafo.</p>

Etiquetas Auto-Cerradas

Algunas etiquetas HTML no necesitan una etiqueta de cierre y se auto-cierran:


					<etiqueta />

Ejemplo:


					<img src="imagen.jpg" alt="Descripción de la imagen" />

Contenido de una etiqueta

De forma general, las etiquetas que tienen contenido pueden tener texto, otra etiqueta o una combinación de texto con otras etiquetas

Las etiquetas HTML pueden anidarse unas dentro de otras para crear estructuras complejas


<div>
	<h1>Título</h1>
	<p>
		Este es un <strong>párrafo importante</strong> dentro de un div.
	</p>
</div>

Atributos

Los atributos se usan para incluir información adicional

Una etiqueta puede tener múltiples atributos


					<etiqueta nombre="valor">Contenido</etiqueta>

Ejemplo:


					<a href="https://www.example.com">Visita Example</a>

Estructura Básica de un Documento HTML


<!DOCTYPE html>
<html>
	<head>
		<!-- Parte no "visible" de la página -->
	</head>
	<body>
		<!-- Parte "visible" de la página -->
	</body>
</html>

Elementos Comunes en HTML

  • <h1> a <h6> - Encabezados
  • <p> - Párrafo
  • <a> - Enlace
  • <img> - Imagen
  • <ul> y <ol> - Listas
  • <div> y <span> - Contenedores

Enlaces y Imágenes


		<a href="https://www.example.com">Este es un enlace</a>
		<img src="ruta/a/imagen.jpg" alt="Descripción de la imagen">

Listas en HTML


		<ul>
			<li>Elemento de lista</li>
			<li>Otro elemento</li>
		</ul>
		
		<ol>
			<li>Primer elemento</li>
			<li>Segundo elemento</li>
		</ol>

Formularios en HTML


		<form action="/submit" method="post">
			<label for="name">Nombre:</label>
			<input type="text" id="name" name="name">
			<br>
			<input type="submit" value="Enviar">
		</form>

Conclusión

HTML es el lenguaje base de la web, y aprenderlo es el primer paso para convertirte en un desarrollador web.
