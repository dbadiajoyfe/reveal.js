Historia de Internet y la Web

David Badía Núñez

Origen de Internet

  • 1969: Nace ARPANET en EE.UU., financiada por el Departamento de Defensa.
  • Motivación: Crear una red robusta para compartir información entre universidades y centros de investigación.
  • ARPANET evoluciona y se expande internacionalmente en los años 80.
  • 1983: Se adopta el protocolo TCP/IP, base de Internet actual.
David Badía Núñez

Origen de la Web

  • 1989-1990: Tim Berners-Lee propone la World Wide Web en el CERN (Suiza).
  • Motivación: Facilitar el acceso y la compartición de información científica entre investigadores.
  • 1991: Se publica el primer sitio web y el primer navegador.
David Badía Núñez

HTML y HTTP

  • HTML (HyperText Markup Language): Propuesto en 1991 por Tim Berners-Lee para estructurar documentos en la web.
  • HTTP (HyperText Transfer Protocol): Protocolo creado en 1991 para la transferencia de información entre servidores y navegadores.
  • Ambos estándares permitieron el crecimiento exponencial de la web.
David Badía Núñez

Impacto y evolución

  • Internet y la web han transformado la comunicación, el acceso a la información y la sociedad.
  • La web ha evolucionado con nuevos estándares y tecnologías (CSS, JavaScript, HTML5, etc.).
